Мама, це моя 1ша татуха

Правила неймінгу:

  • малий регістр
  • без пробілів
  • без спец символів

Ділимо tag-и на умовні категорії

  1. Будівельні теги:
    • div
    • header
    • main
    • footer
  2. Контекстні теги:
    • p = параграф тексту (блочний)
    • a = посилання (рядковий)
    • img = картинка (рядковий)
    • br = обрив рядка (МАЙЖЕ блочний)
    • span = виділення частини рядка (рядковий)
    • mark = спан + дефолт стилі (рядковий)
    • ul = маркований список (блочний)
    • ol = нумерований список (блочний)
    • li = пункт списку ol/ul (блочний)
    • h1,h2... = заголовки (#=value) (блочний)
